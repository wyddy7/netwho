# Epic 4: Telegram Бот - Голосовой Пайплайн (MVP Feature 1)

**Цель:** Собрать всё воедино в интерфейсе Telegram. Обработка голосовых сообщений.

## Шаг 4.1: Базовый бот
- [ ] Создать `app/main.py` (entrypoint).
- [ ] Инициализировать `Bot` и `Dispatcher`.
- [ ] Настроить polling.

## Шаг 4.2: Команды Start и Help
- [ ] Создать `app/handlers/base.py`.
- [ ] Реализовать `/start`:
  - [ ] Регистрация пользователя в БД.
  - [ ] Вывод оферты (Inline кнопка "Согласен").
  - [ ] Сохранение статуса `terms_accepted`.
- [ ] Реализовать `/help`.

## Шаг 4.3: Обработчик голосовых (Voice Handler)
- [ ] Создать `app/handlers/voice.py`.
- [ ] Реализовать pipeline обработки:
  1. Скачивание файла (`bot.download`).
  2. Конвертация (`audio_service`).
  3. Транскрибация (`ai_service`).
  4. Экстракция сущностей (`ai_service`).
  5. Генерация вектора.
  6. Сохранение в БД (`search_service`).
- [ ] Отправка ответа пользователю (отформатированный текст + теги).
- [ ] Обработка ошибок (сообщения пользователю, если Whisper/LLM упали).

**Критерий готовности:**
- Бот запускается. Можно отправить голосовое "Дима, директор завода, любит рыбалку", и бот ответит "Сохранил: Дима (директор завода)". Запись появляется в БД.
